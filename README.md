# 🏠 City Real Estate

A modern, corporate real estate platform built with pure HTML, CSS, and JavaScript — featuring property listings, search filters, and a Firebase-powered inquiry form.

## ✨ Features

- ✅ Property search box with purpose, location, type, budget filters
- ✅ Filterable property listings (Sale / Rent / Plots / Commercial)
- ✅ Property cards with beds, baths, area, status badges
- ✅ Firebase Firestore inquiry form
- ✅ Agent contact card with WhatsApp
- ✅ Modern navy + sky blue corporate design
- ✅ Fully responsive
- ✅ GitHub Pages ready

## 🔧 Customization

| What to change | Where |
|---|---|
| Company name | Search `City` / `CityEstate` → replace |
| Properties | Find `listings-grid` section |
| Phone / email | Search `+92 309 6455356` → replace |
| Agent name | Find `agent-card` section |
| Colors | Edit `:root` CSS variables |
